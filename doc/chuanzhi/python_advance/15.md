## udp((User Datagram(数据报) Protocol) 通讯

### 使用方式对比手机发短息即可(很多可以类比于此)

- 拿起手机
- 发送输入的信息
- 放下手机

```sh

import socket

'''
 create socket

'''

remote_address = ('192.168.12.250', 8080)

upd_socket = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)

upd_socket.sendto(('nice boy'*1000).encode(), remote_address)

upd_socket.close()

```
