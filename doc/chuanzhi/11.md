## 一切皆对象，类 10,.. 都是由对象创建的

```

x.__class__ # 获取创建这个对象的类

a = 10
a.__class__  # int

# 元类 会用的时候 就会用了 不会的时候就不会
# 类也是对象  __class__ 为 type（元类，所有的类都是由它创建的）

```
